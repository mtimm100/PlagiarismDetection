User's manual for Mikayla Timm's Project 1:

SETUP:
Put all text files less than or equal to 20000 characters in length into 
/datafiles/ subfolder of Project1DSA2. Maximum number of files allowed is 30. Maximum word length is 30. Maximum file name length is 50. 

HOW TO RUN:
cd to Project1DSA2
gcc Hash.c Node.c FileIO.c StringProcessing.c ComparingFilesTest.c
./a.out

Enter in a number (2-10) for phrase length to see file similarity.
